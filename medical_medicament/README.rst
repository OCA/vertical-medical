==================
Medical Medicament
==================

This module introduces the concept of medicaments, using product as a basis. It
also adds drug forms, drug routes, and medicament components.

Please note that medicament storage info is no longer part of this module and
can be found in a more reusable form in ``medical_medicament_storage``.

Usage
=====

#. Go to ``Medical`` -> ``Configuration`` -> ``Medicaments``
#. Select ``Medicaments`` to review, add, or modify medicaments
#. Select ``Admin Routes`` to review, add, or modify drug administration routes
#. Select ``Components`` to review, add, or modify medicament components
#. Select ``Forms`` to review, add, or modify drug forms
