===================================
Website Sale - Medical Prescription
===================================

This module binds website_sale and medical_prescription_order

Installation
============

To install this module, you need to:

* Install module

Configuration
=============

To configure this module, you need to:

* Go to Website Admin for Pharmacy website
* Enable ``Prescription Step Option`` in Website Checkout customization

Usage
=====

To use this module, you need to:

* Buy product that is child of Prescription internal category using website
